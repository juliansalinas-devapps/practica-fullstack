# Frontend - Angular Application

Interfaz de usuario profesional diseñada para la gestión intuitiva de productos.

## Tecnologías
- **Angular 19**: Framework de desarrollo.
- **Bootstrap 5**: Estilos y componentes responsivos (Modales, Tablas).
- **Reactive Forms**: Para validaciones de entrada en tiempo real.
- **HttpClient**: Comunicación asíncrona con el backend de Spring.

## Componentes Clave
- **Tabla de Inventario**: Visualización dinámica con badges de estado.
- **Modal de Gestión**: Formulario inteligente compartido para crear y editar.
- **Ajustes de Stock**: Integración de prompts para movimientos rápidos de inventario.

## Ejecución
1. Instalar dependencias: `npm install`
2. Iniciar servidor: `ng serve`
