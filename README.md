# Práctica Full-Stack Jr - Control de Inventarios

Este proyecto es una aplicación Full-Stack para la gestión de inventarios, desarrollada como parte de una práctica técnica profesional.

## Tecnologías Principales
* **Backend:** Java 17, Spring Boot 3.x, Spring Data JPA.
* **Base de Datos:** MySQL (XAMPP).
* **Frontend:** Angular (Próximamente).
* **Control de Versiones:** Git & GitHub.

## Estructura del Repositorio
* `/backend`: Código fuente de la API RESTful.
* `/frontend`: Aplicación cliente en Angular (En construcción).
* `/docs`: Documentación y diagramas del modelo ER.
